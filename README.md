# BodyType-Trainer 1 Alpha
05/07/2026 - Santiago Hurtado
05/07/2026 - Revisado por Alejandro Rodriguez (COLDEX)

BodyType Trainer es una aplicación de escritorio que permite calcular diferentes indicadores de composición corporal, visualizar resultados de forma intuitiva y almacenar el historial de cada usuario mediante una base de datos.

Trabajado con:
Python
SQLite

Versiones:
python 3.14.6
SQLite 3.53.0

Instalar dependencias:
pip install requeriments.txt

Ingresar base de datos:
/ld nombre.db

Iniciar python:
python main.py
